Example of Auth

Single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
  • PostgreSQL 13 relational database
  • Swagger

Details of the project:

  • In order to run the project use the command: dotnet run
  • Regarding the frontend it can be runned seperate ng serve. It runs in the background automatically, so your client-side resources are dynamically built on demand and the page refreshes when you modify any file.
  • Backend auth implemented using identity.
  • Token saved on localstorage as __TOKEN__.
